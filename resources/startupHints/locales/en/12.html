La bibliothèque possède un dossier "Favoris" dans lequel il est possible de stocker des références Web.

Un glisser-déposer insère la page Web dans un widget.

Un clic sur l'icône ouvre la page Web dans le navigateur d'Open-Sankoré.

L'ajout d'un favori se fait par le bouton "Signet" dans la barre d'outils du mode Web.
