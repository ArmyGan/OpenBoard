Une capture de contenu Web est désormais élargie sur la version Open-Sankoré 2.10.

Cet outil est disponible dans l'onglet Web aux côtés de capturer une partie de l'écran, capturer une fenêtre, afficher le clavier virtuel.

 

Cet vous permet de capturer le contenu le plus courant sur une page Web. Cette capture peut être de deux formes :

Stockage du contenu directement dans la bibliothèque ou la page

Création d'un lien dans la bibliothèque ou sur une page si les droits d'auteurs ne vous le permettent pas. Un accès Web sera alors requis pour lire ensuite le document.
